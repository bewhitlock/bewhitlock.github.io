Ball-On-Plate

This project was my first introduction to the magic of PID and Machine Learning! I made a python program to run on a raspberry pi. It tracks the ball, and controls the plate angle. Using this framework, I tested both PID and a machine learning algorithm called Soft-Actor-Critic. Preliminary findings show that SAC is better at dynamic trajectory tracking, but both PD and SAC are acceptable methods in nonlinear control systems. My favorite part about this project was learning about SAC and getting experience with the machine learning library Gymnasium which allows you to build virtual simulations and train machine learning models for real-world scenarios. I feel like its something I can apply to another project in the future...
